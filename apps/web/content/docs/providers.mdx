# Providers

Archflow supports multiple cloud and infrastructure providers with official icon sets.

| Provider | Icons | Version |
|----------|-------|---------|
| AWS | 307 nodes, 11 clusters | 2025-07-31 |
| Azure | 626 nodes | November 2025 |
| GCP | 19 nodes | 2025-09 |
| Kubernetes | 39 nodes | 2022-05-15 |

## Usage

```
use aws
use azure
use gcp
use k8s
```

## Custom Sources

```
use my-provider from github:org/repo
```
